Accept payments securely through MaishaPay on WooCommerce using Mobile Money and bank cards.

== Description ==

MaishaPay Gateway for WooCommerce allows merchants to securely accept online payments through the MaishaPay payment platform.

Supported payment methods include:

* MPESA
* Airtel Money
* Orange Money
* Visa
* Mastercard
* American Express

The plugin redirects customers to MaishaPay's secure payment page where they can complete their transaction safely.

== Features ==

* Secure payment gateway
* Sandbox and Live modes
* Mobile Money payments
* Credit and debit card payments
* Automatic payment confirmation
* WooCommerce order synchronization
* Dashboard with transaction statistics
* Transaction history
* Reports page
* Automatic checkout page compatibility fix
* HPOS (High Performance Order Storage) compatible
* Supports multiple currencies: USD, CDF, CFA, EUR

== Requirements ==

* WordPress 6.0 or higher
* WooCommerce 8.0 or higher
* PHP 7.4 or higher
* A MaishaPay merchant account

== Installation ==

1. Download the plugin ZIP.
2. Go to WordPress Admin > Plugins > Add New.
3. Click Upload Plugin.
4. Select the ZIP file.
5. Install and activate the plugin.
6. Go to WooCommerce > Settings > Payments.
7. Enable "MaishaPay".
8. Enter your Public API Key and Secret API Key.
9. Choose Sandbox or Live mode.
10. Save your settings.

== Configuration ==

Create your merchant account on MaishaPay.

After receiving your API keys:

* Enable the payment gateway.
* Choose the transaction currency.
* Configure Sandbox or Live mode.
* Enter your Public API Key.
* Enter your Secret API Key.

Your customers can then pay using Mobile Money or bank cards.

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =

Yes. WooCommerce must be installed and activated.

= Is Sandbox mode available? =

Yes. You can test payments using Sandbox mode before going live.

= Which currencies are supported? =

USD, CDF, CFA and EUR.

= Is HPOS supported? =

Yes. The plugin is compatible with WooCommerce High Performance Order Storage (HPOS).

= Can I accept card payments? =

Yes. Visa, Mastercard and American Express are supported.
