# MaishaPay Gateway for WooCommerce

Official WooCommerce payment gateway developed by **MaishaPay**.

**Author:** Jacques Mvumbi

Accept payments directly on your WooCommerce store using:

- Mobile Money
- Visa
- Mastercard
- Bank Cards
- Wallet Payments

---

## Features

- Easy installation
- Secure Checkout
- Mobile Money support
- Visa & Mastercard payments
- Sandbox mode
- Live mode
- Webhook support
- Automatic payment confirmation
- Refund API support
- WooCommerce HPOS compatible
- Secure REST API integration
- Automatic order status updates

---

## Requirements

- PHP 7.4 or higher
- WordPress 6.4 or higher
- WooCommerce 8.0 or higher

---

## Installation

1. Download the latest release.
2. Upload the plugin to `/wp-content/plugins/`.
3. Activate **MaishaPay Gateway** from the WordPress Plugins page.
4. Navigate to **WooCommerce → Settings → Payments**.
5. Enable **MaishaPay Gateway**.
6. Enter your API credentials.
7. Save your settings.

---

## Configuration

You'll need the following credentials from your MaishaPay Merchant Dashboard:

- API Key
- Secret Key

Once configured, your customers will be able to pay securely using Mobile Money and bank cards.
